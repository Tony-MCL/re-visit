# re-visit (working title)

A private mobile app for capturing real experiences:
- take one photo
- auto save time + GPS
- answer: did I like this? (Yes / Neutral / No)
- optional short comment
- builds a personal, honest log

## Run
1) Install dependencies:
   npm install

2) Start:
   npx expo start

## Notes
- This repo intentionally does not use lock files.
- Data is stored locally on-device with AsyncStorage.
